1.Трябва да имате VS Code и SQL Server изтеглени на вашия компютър
2.Отваряме проекта чрез VS Code
3.Отваряме 2 терминала
4.В единия терминал влизаме в папката сървър чрез командите: cd server-> node server.js
5.В другия терминал пишем npm start
6.Отваряме сайта като отидем в браузъра и напишем localhost:3000
7.Всичко е готово! :)
